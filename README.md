# TO-DO-Notes-app

## Description 
My application is going to be a minimul notes application that forfulls the a teams workflow and but is also easy to use and elegant for a single user. It will allow for easy to manage workflow. But also include an accesible area for TO-DO's for life in general. It will be customisable and great for developers that want to save code for later with the easy to read syntax highlighting for: python, javascript, css , scss and html.

My application is going to be a web app. This will allow the fluidity of devices with very minimul effort. It will also be a built on react to help with the layout and the dynamic content. 

## Functionality brief:
- [ ] Delete items
- [ ] Add Items
- [ ] Markdown
- [ ] Device Fluid
- [ ] Checklist capability
- [ ] file attachments 
- [ ] Tags
- [ ] Scheduled Reminders
- [ ] Daily Agenda
- [ ] Connected to google calendars 
- [ ] Section for general TO-DO's and a section for projects 
- [ ] Each project will also have a section for Task / TO-DO's and a section for notes
- [ ] Task / TO-DO section it will be split into "Zones" which will be area to structure put notes like a folder. These might be like "ideas", "active", "done" etc 
- [ ] Every list of notes should have the ability to pin notes
- [ ] Sort notes by Date Created, Custom, Last Edited, etc
- [ ] Optional Colours
- [ ] Light or Dark modes
- [ ] Email notifications
- [ ] Push notifications
- [ ] Swipe features on each note/task. Such as move to..., delete, etc.
- [ ] Syntax Highlighting
- [ ] Share with users

### Storage
I will have an option to save storage locally and another option for saving the data on the cloud for access from the an acount on both the computer and on the phone.
